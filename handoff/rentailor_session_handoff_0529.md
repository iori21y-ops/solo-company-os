## 4. 이번 세션 확정 결정사항

### 재미요소 모델 전환
- 듀오링고(스트릭·게임화) ❌ → **토스식 Zero Jargon**(쉬운 용어·비유) ✅
- 뱅크샐러드식 숫자 시각화. 게임화는 2차.
- **법령용어 왜곡 금지** — 정식 중개업 등록 = 토스 규제 사례(외상/홀짝)와 동일 리스크.

### 시나리오 콘텐츠 방향
- 모토: "이 플랫폼에서 고객의 모든 문제가 해결된다."
- 모든 시나리오는 **상황→비교→렌테일러에서 해결** 3부로 반드시 닫힘. 정보글로 끝나는 것 0개.
- 캐릭터: 질문형("잔존가치가 뭐예요?")이 아니라 **상황형**("출퇴근이 길어서 약정거리 넘을 것 같은데")으로. 차알못 검색 유입에 강함.

### 캐릭터 14명 확정
- 최상위 구분: **개인(6)/개인사업자(4)/법인(4)** — 제도적 처지(세무·명의)가 여정의 답을 실제로 가르기 때문.
- 제작 효율: 사건은 공유 자산(1번만 작성), 캐릭터는 "그 사건에 대한 답"만 다르게. 답이 같은 캐릭터는 묶어서 반복 제거.
- CARTYPE(4축 16유형)는 별도 트랙(성향 진단용). 캐릭터 여정과 혼용 금지.

### 숏츠 방향
- 토스피드식 일상 스토리 → **자막+차량이미지+TTS** 숏츠로 제작.
- 목적: 광고 수익 아님. **플랫폼 유입 파이프.** 알고리즘으로 신규 채널도 1만+ 조회 가능.
- 인프라: 대본(사건풀)·차량이미지(car-images/360)·FFmpeg·맥미니 TTS로 추가비용 0원.
- **1개를 수동으로 끝까지** 만들어 반응 보기 → 검증 후 n8n 자동화. 이전 합의(v2 보류, 5~10회 검증 후)와 정합.
- 첫 숏츠 제작 시작함 (이 세션 마지막 작업).

## 4-1. 최종 확정 — 동물 캐릭터 = 고객 아이덴티티 (재검토 없이 실행)

**확정:** 고객이 본인에게 맞는 동물 캐릭터를 고르고, 그 캐릭터에 맞는 콘텐츠·추천·케어를 받는 구조.
- 동물 캐릭터 = 단순 콘텐츠 소재가 아니라 **서비스 전체의 고객 아이덴티티**.
- 마이페이지 아이콘, 위키·숏츠 여정 스토리, 케어 푸시, SNS 공유까지 하나로 관통.
- 진입: 진단 → 동물 배정 → 그 동물의 콘텐츠/케어/재계약 경험.
- 동물 수: 최소 14마리(현재 캐릭터 기반), 처지 축 추가 시 확장 가능.
- 답 분기 묶음 구조 덕분에 동물이 늘어도 콘텐츠가 비례 증가하지 않음.
- CARTYPE 진단 → 동물 배정 로직으로 연결 (Phase 2에서 설계).
- **경쟁사 방어벽:** 숏츠 콘텐츠는 카피 가능하지만, "동물=고객아이덴티티=서비스 전체 경험"은 시스템이라 카피 불가.

**캐릭터 서비스 6가지 패턴 (성공사례 기반, 적용 확정):**

| 순위 | 패턴 | 참고 사례 | 렌테일러 적용 | Phase |
|---|---|---|---|---|
| 1 | 프로필 아이콘 = 내 정체성 | 카카오프렌즈/라인프렌즈 | 마이페이지·케어알림·견적결과지에 내 동물 아바타 | 1 |
| 2 | 스토리 유니버스(세계관) | 카카오 홍이, 라인프렌즈 | 14마리가 서로 아는 사이. 크로스오버 숏츠("고슴도치가 여우에게 물어봄") | 1 |
| 3 | 활동 뱃지/악세서리 | 당근마켓 배지(희소성 강조) | 견적완료·비교마스터·케어확인 뱃지 → 캐릭터에 모자·망토 착용으로 시각화 | 2 |
| 4 | 진화/성장 시스템 | 다마고치(50종 진화) | 계약전=알→견적=아기→계약=성체→재계약=진화형. 처지 변화 시 동물 종 변경 가능 | 2 |
| 5 | 금융학습=캐릭터활동 | 퓨처뱅크 삐뽀 | 위키읽기=경험치, 견적비교=아이템, 케어확인=행복도↑ | 2 |
| 6 | 굿즈/콜라보/IP확장 | 카카오프렌즈(6400종 상품) | 계약 완료 시 내 캐릭터 차량 스티커 증정. 인지도 후 자동차 굿즈 콜라보 | 3 |

**Phase 구분:**
- Phase 1: 동물 14마리 이미지 AI 생성 + 숏츠 10~15개 제작·예약 업로드 + 프로필 아이콘 + 스토리 유니버스 기반 콘텐츠
- Phase 2: 진단→동물 배정→마이페이지→케어 연결 + 뱃지/악세서리 + 진화/성장 + 학습=육성 시스템
- Phase 3: 등급·SNS 공유 + 굿즈/콜라보/IP 확장

**동물 매핑 (현재 14마리):**
P1 고슴도치(첫차초년생) / P2 백조(워킹맘) / P3 여우(트렌드) / P4 거북이(시니어) / P5 다람쥐(대출앞둔직장인) / P6 말(장거리출퇴근) / S1 시바견(영업사업자) / S2 고양이(자영업) / S3 카멜레온(프리랜서) / S4 사자(고소득전문직) / C1 곰(중소기업대표) / C2 개미(플릿관리) / C3 병아리(스타트업법인) / C4 코끼리(복지차량)

## 4-2. 캐릭터 성공사례 분석 + 적용 계획

**참고한 성공사례 5개:**

| 사례 | 핵심 성과 | 렌테일러 적용 |
|---|---|---|
| 하이트진로 두꺼비 | 유튜브 4,203만뷰, 굿즈 90초 품절, 140종 콜라보 | 캐릭터가 금융정보를 대신 말함. 장기적으로 딜러 매장 캐릭터 존 |
| 펭수 | 수입 100억+, 전세대 인기 | 14마리 각자 고유 말투 부여 → 숏츠 TTS 톤 차별화 |
| 망그러진곰/최고심 | 이모티콘→IP 성장, 인스타 50만/20만 팔로워 | 카카오 이모티콘 출시 = 무료 바이럴 채널. 직장인 공감형 |
| KB국민카드×최고심 | 캐릭터 콜라보로 카드 사용량 2배 | 캐피탈사와 "캐릭터 전용 프로모션" 콜라보 가능 |
| 대구 뚜비 | 10개월 만에 굿즈 매출 1억, 뮤지컬 전회차 매진 | 계약 완료 고객에 "내 동물 차량 스티커/방향제" 증정 → 실물 바이럴 |

**캐릭터가 먹히는 5가지 조건 (사례 공통):**

| 조건 | 설명 | 렌테일러 체크 |
|---|---|---|
| ① 고유 인격 | 말투·성격·대사 | 14마리 각자 말투 설정 ✅ |
| ② 세계관 | 캐릭터끼리 관계·스토리 | 크로스오버 대화 숏츠 ✅ |
| ③ 접점 다양화 | 이모티콘·굿즈·영상·SNS | 숏츠+이모티콘+계약굿즈 ✅ |
| ④ 수집/희소 | 모으고 싶게, 한정판 | 뱃지·진화·숨겨진 캐릭터 ✅ |
| ⑤ "나"와 연결 | 내 캐릭터라는 정체성 | 진단→배정→마이페이지 아이콘 ✅ |

**캐릭터 서비스 확장 로드맵 (Phase별):**
- Phase 1: 동물 이미지 + 고유 말투 설정 + 숏츠(세계관 포함 크로스오버)
- Phase 2: 뱃지/악세서리 + 진화/성장 + 학습=육성 + 이모티콘 출시
- Phase 3: 굿즈(차량 스티커/방향제) + 캐피탈사 콜라보 프로모션 + IP 확장
