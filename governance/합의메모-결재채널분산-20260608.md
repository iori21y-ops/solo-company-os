# 합의메모 — 결재 채널 분산 전환 (2026-06-08)

## 결정 사항

**모든 결재판은 해당 에이전트의 전용 채널에서 직접 제출한다.**  
대표이사실(`#대표이사실`)로 결재를 집중하는 방식 폐지.

## 배경

기획팀장(OpenClaw)이 데이터·콘텐츠·마케팅 결재를 대표이사실에 집중 제출해왔음.  
편리하나 채널 혼잡 및 맥락 혼선 발생 → 대표 지시로 분산 전환.

## 적용 규칙

| 에이전트 | 결재 제출 채널 |
|---------|-------------|
| 데이터 | `#데이터` |
| 콘텐츠 | `#컨텐츠` |
| 마케팅 | `#마케팅` |
| 기획팀장 | 해당 업무 채널 (데이터/콘텐츠/마케팅 중 관련 채널) |
| 관리팀장 | 해당 업무 채널 또는 `#관리` |

## 대표이사실 사용 범위 (변경 후)

- 대표 최종 결정이 필요한 **에스컬레이션**만 사용
- 일일 요약 보고
- 대표 직접 지시 수신

## 변경된 문서

- `REG-001-결재보고프로세스.md` v1.1 — 10조 추가

## 기획팀장 전달 필요 사항

- `[APPROVAL:대표님]` 카드를 대표이사실로 보내지 않는다
- 데이터 결재 → `#데이터`, 콘텐츠 결재 → `#컨텐츠`, 마케팅 결재 → `#마케팅`에서 처리
- 대표이사실에는 일일 보고서·에스컬레이션만 올린다
