# 데이터 신선도 임계값

## 운영 원칙

- 이 파일은 에이전트 운영 설정. 학습 페이지(wiki)와 분리.
- audit 사이클 누적에 따라 규칙/임계 보정 가능 (사람 승인 필요).
- 변경 시 사이클 프론트매터 `rules_applied`에 영향 명시.
- **WATCH**: 경과/임계 ≥ 50% → 다음 A-2에서 모니터링 강화.
- **RED**: 경과 > 임계 → 수집 즉시 요청, downstream 사용 금지.
