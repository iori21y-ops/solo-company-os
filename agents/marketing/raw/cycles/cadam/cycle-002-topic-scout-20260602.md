# cycle-002 — 주기B 주제 발굴 2026-06-02

## 1. 배경

- 트리거: 관리팀장 지시, 주기 업무 자동 진행 착수
- content 에이전트 스카우트 후보 미수신 (첫 주기B, content와 협업 루프 미개통)
- 자체 트렌드 + 시즌 신호(6월: 초여름·반기결산·휴가)로 후보 5건 발굴
- 기대 결과: Tier 1 주제 2건 이상 → 발행계획 확정(주기A·content 주기1 시작 조건)

## 2. 입력

- 시즌 신호: 6월 초여름 / 7월 휴가 준비 / 상반기 결산(법인차 수요)
- CADAM 포지셔닝: 장기렌트 비교 플랫폼, 첫 상담 고객 유입 중심
- 현재 보유 데이터 자산: 아반떼 이미지 verdict green
- 참조 sales 전환 데이터: 없음 (첫 사이클, Q4는 추정)

## 3. 주제 후보 5건 Tier 판정

### 후보①: 현대 아반떼 장기렌트 월 납입금 얼마?
- Q1(사업적합도): **yes** — 아반떼 data 보유, CADAM 핵심 포지셔닝
- Q4(전환력): **yes** — 가격 정보 탐색 = 구매 의사 고객, 상담 신청 1-step
- Q5(타깃일치): **[Q5 게이트1 — 사람 확인 필요]** — 아반떼 = 소형차 타깃(20-30대). CADAM 주요 타깃과 일치하는지 대표 확인 요망
- 판정: Q1+Q4 yes, Q5 pending → **잠정 Tier 1** (Q5 확인 후 확정)
- 채널: 네이버 블로그 SEO (검색 유입 최우선)

---

### 후보②: 장기렌트 vs 할부 — 2026년 어느 게 유리할까
- Q1(사업적합도): **yes** — "비교"는 CADAM의 핵심 가치 제안과 직결
- Q4(전환력): **yes** — 비교 탐색 단계 고객 유입 → CADAM 비교 페이지 CTA
- Q5(타깃일치): **[Q5 게이트1 — 사람 확인 필요]** — 광범위 타깃, CADAM 주요 퍼널에 맞는지 확인 요망
- 판정: Q1+Q4 yes, Q5 pending → **잠정 Tier 1**
- 채널: 자사 블로그 + 인스타 요약카드

---

### 후보③: 여름 휴가 전 차 준비, 장기렌트 지금 신청해도 될까?
- Q1(사업적합도): **yes** — 시즌성 캠페인, 즉각 행동 유도
- Q4(전환력): **yes** — "지금 신청"이라는 행동 의사 있는 독자
- Q5(타깃일치): **[Q5 게이트1 — 사람 확인 필요]** — 시즌 타깃이 CADAM 주요 고객층과 일치하는지 확인
- 판정: Q1+Q4 yes, Q5 pending → **잠정 Tier 1**
- 채널: 인스타그램 + 카드뉴스 (시즌 임박성 활용)

---

### 후보④: 법인차 장기렌트, 상반기 결산 전에 알아야 할 것
- Q1(사업적합도): **[Q1 게이트1 — 사람 확인 필요]** — CADAM이 법인 고객을 현재 주요 타깃으로 보는지 확인 필요
- Q4(전환력): **yes** — 6월 반기결산 시즌, 법인차 수요 집중
- Q5(타깃일치): **[Q5 게이트1 — 사람 확인 필요]** — B2B 타깃 여부 확인
- 판정: Q4만 yes → **잠정 Tier 2** (Q1·Q5 모두 pending)
- 채널: 네이버 블로그 SEO

---

### 후보⑤: SUV 장기렌트 비교 — 가족 차 어떤 게 좋을까?
- Q1(사업적합도): **yes** — SUV 비교는 CADAM 컨텐츠 카테고리 자연스럽게 확장
- Q4(전환력): **yes** — 가족 차 탐색 = 구매 결정 임박 고객 비율 높음
- Q5(타깃일치): **[Q5 게이트1 — 사람 확인 필요]** — SUV/가족차 타깃이 CADAM 현재 주요 고객인지 확인
- 판정: Q1+Q4 yes, Q5 pending → **잠정 Tier 1**
- 채널: 네이버 블로그 SEO + 인스타 카드뉴스
