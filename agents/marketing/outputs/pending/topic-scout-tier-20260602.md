## 사람 확인 필요 Q&A (게이트1)

```

Q④-1: CADAM이 법인 고객을 현재 주요 타깃으로 추진하는가?               [yes / no]
Q⑤-5: SUV/가족차 카테고리를 지금 콘텐츠 우선순위로 볼 것인가?          [yes / no]
```

## 게이트1 승인 후 확정 발행계획

Q 확인 결과에 따라 확정되는 즉시 주기A·content 주기1 시작 조건 충족.

- 아반떼 Q①-5 yes → **다음 주기A 1순위**: 아반떼 장기렌트 네이버블로그
- 여름시즌 Q③-5 yes → **다음 주기A 2순위**: 여름휴가 인스타 카드뉴스 (7월 전 발행 타겟)
- 법인차 Q④-1 no → 후보⑤ 보류, 나머지로 집중
