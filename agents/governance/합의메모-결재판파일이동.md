## 확정 사항
1. 명령어(사장님 전용, 봇 차단):
   - `대기 <에이전트>`  → outputs/pending/ 파일 목록 반환
   - `승인 <에이전트> <파일명>` → pending → approved (renameSync)
   - `반려 <에이전트> <파일명>` → pending → rejected (renameSync)
2. 경로: ~/projects/_meta/agents/<에이전트>/outputs/<상태>/
3. import 추가: agent.js 2줄 fs import에 renameSync, readdirSync 추가
4. 핸들러 위치: 복구승인 블록 직후 ~ `if (message.author.bot)` (112줄) 직전
5. 149줄 [승인]/[반려] 봇 completed 플래그는 건드리지 않음 (사람 결재와 분리)

  (요약 아닌 원문을 읽고 맞출 것 — 추측 금지)

- 없는 파일/없는 에이전트 폴더 → 에러 메시지 반환, 크래시 금지
