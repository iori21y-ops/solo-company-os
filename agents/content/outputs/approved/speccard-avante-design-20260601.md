## 4. 비주얼 스타일

### 컬러
| 역할 | 컬러값 |
|------|--------|
| 배경 | `#0D0F1A` (딥 네이비) |
| 텍스트 주 | `#FFFFFF` |
| 텍스트 보조 | `#A0A8C0` |
| 스펙 라벨 | `#6B74A0` |
| 강조 (아이콘) | `#4FC3F7` (라이트 블루) |
| 구분선 | `#1E2238` |
| 트림 뱃지 배경 | `#1A2F4A` |
| 풋터 배경 | `#080A12` |

### 타이포그래피
| 요소 | 서체 | 크기 | 굵기 |
|------|------|------|------|
| 모델명 | Pretendard | 52px | Bold (700) |
| 트림명 | Pretendard | 24px | Medium (500) |
| 스펙 수치 | Pretendard | 32px | SemiBold (600) |
| 스펙 라벨 | Pretendard | 18px | Regular (400) |
| 풋터 | Pretendard | 16px | Regular (400) |

### 이미지 처리
- 히어로 이미지: 측면 3/4 컷, 우측 향 (차량 진행 방향 → 우)
- 배경 제거 후 딥 네이비 위에 배치
- 하단에 레이어드 그림자 (box-shadow 느낌) + 미러 반사 효과 (20% 투명도)
- 이미지 소스: WF-VI referential green 승인 자산 사용 (data verdict: `verdict-hyundai-avante-20260601.md`)

### 스펙 그리드 처리
- 각 셀: 좌측 아이콘(24px) + 우측 텍스트
- 셀 구분: `#1E2238` 1px 선
- 홀수 행: 배경 `#0F1220` / 짝수 행: 배경 `#0D0F1A`
