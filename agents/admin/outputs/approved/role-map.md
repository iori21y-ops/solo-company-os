# 운영 역할 정의서

작성일: 2026-05-30  
문서 목적: 데이터팀장, 컨텐츠팀장, 마케팅팀장의 역할과 업무 흐름을 명확히 구분해 위임 기준으로 사용한다.

## 1. 전체 구조

세 역할은 하나의 콘텐츠 생산 라인으로 연결된다.

```text
데이터팀장
  사실과 원재료 수집
        ↓
컨텐츠팀장
  사람이 보기 좋은 자료로 가공
        ↓
마케팅팀장
  외부 채널에 발행하고 유입 확보
        ↓
대표님
  기준 설정, 최종 판단, 승인
```

한 줄로 정리하면 데이터는 사실, 컨텐츠는 표현, 마케팅은 확산을 맡는다.

## 2. 역할별 정의

### 데이터팀장

데이터팀장은 모든 콘텐츠와 마케팅의 원재료를 수집하고 검증하는 역할이다.  
핵심은 많이 모으는 것이 아니라, 믿고 쓸 수 있는 상태로 정리하는 것이다.

주요 업무:

- 크롤링 및 원천 데이터 수집
- DB 정리와 필드 기준 관리
- 시장조사와 경쟁사 조사
- 상품, 브랜드, 키워드, 가격 등 기초 정보 검증
- 360 이미지, 흰배경 이미지, 홈카드 이미지 등 표준 표시 자산 관리

주요 산출물:

- 검증된 원천 데이터
- 정리된 DB
- 시장조사 자료
- 상품 및 브랜드 기준 정보
- 표준 이미지 자산

성과 기준:

- 데이터 정확성
- DB 정합성
- 최신성
- 누락률
- 업데이트 적시성

위임 기준:

- 반복 수집, 정리, 비교 업무는 위임 가능
- 검증 기준과 판단 로직은 사람이 먼저 설계해야 함
- 애매한 데이터는 임의 판단하지 않고 확인 대상으로 올려야 함

## 3. 컨텐츠팀장

컨텐츠팀장은 데이터팀장이 만든 원재료를 사람이 이해하기 쉬운 자료와 에셋으로 바꾸는 역할이다.  
핵심은 보기 좋게 만드는 것보다, 정확한 정보를 재사용 가능한 형태로 가공하는 것이다.

주요 업무:

- 인포그래픽 제작
- 비교표 제작
- 차트와 시각 자료 제작
- 팩트블록 구성
- 인사이트 리포트 작성
- 내부 자료를 카드뉴스, 블로그, 랜딩 등에 재사용 가능한 에셋으로 정리

주요 산출물:

- 콘텐츠 에셋
- 비교 콘텐츠
- 설명 자료
- 차트 및 시각화 자료
- 인사이트 리포트
- 마케팅팀장이 활용할 수 있는 원고 재료

성과 기준:

- 에셋 재사용률
- 가독성
- 데이터 오류율
- 제작 속도
- 채널별 변환 용이성

위임 기준:

- 초안 제작과 에셋 가공은 위임 가능
- 숫자, 사실, 비교 기준은 반드시 검수 필요
- 출처가 불명확한 내용은 콘텐츠에 포함하지 않음

## 4. 마케팅팀장

마케팅팀장은 완성된 콘텐츠 에셋을 외부 채널에 맞게 발행하고, 유입과 전환을 만드는 역할이다.  
핵심은 콘텐츠를 예쁘게 내보내는 것이 아니라, 목표 고객이 클릭하고 행동하게 만드는 것이다.

주요 업무:

- SEO 블로그 글 작성
- 카드뉴스 문안 구성
- Shorts 및 숏폼 대본 작성
- 광고 카피 작성
- 랜딩페이지 카피 작성
- 채널별 제목, 후킹 문구, CTA 설계

주요 산출물:

- 발행용 콘텐츠
- 광고 문안
- 숏폼 대본
- 검색 유입용 글
- 랜딩 카피
- 캠페인 메시지

성과 기준:

- 도달
- 클릭률
- 유입
- 리드 전환
- 구매 또는 상담 전환

위임 기준:

- 초안 작성, 제목 후보, 카피 후보 생성은 위임 가능
- 최종 발행은 사람 검토 후 진행
- 브랜드 톤, 법적 표현, 과장 광고 가능성은 반드시 확인

## 5. 업무 연결 기준

### 데이터팀장에서 컨텐츠팀장으로 넘길 때

전달해야 할 것:

- 원천 데이터
- 출처
- 수집 기준
- 검증 여부
- 주의해야 할 해석

넘기면 안 되는 상태:

- 출처가 없는 수치
- 기준이 불명확한 비교
- 최신 여부가 확인되지 않은 정보
- 오류 가능성이 표시되지 않은 자료

### 컨텐츠팀장에서 마케팅팀장으로 넘길 때

전달해야 할 것:

- 핵심 메시지
- 시각 자료
- 비교 포인트
- 고객에게 강조할 장점
- 사용 가능한 문장 재료

넘기면 안 되는 상태:

- 검증되지 않은 주장
- 맥락 없는 이미지
- 고객 관점으로 정리되지 않은 긴 설명
- 채널 목적과 맞지 않는 자료

### 마케팅팀장에서 대표님에게 올릴 때

전달해야 할 것:

- 발행 초안
- 제목 후보
- 핵심 후킹 문구
- CTA
- 예상 리스크
- 검토가 필요한 표현

대표님이 판단할 것:

- 최종 발행 여부
- 브랜드 톤 적합성
- 리스크 수용 여부
- 우선순위
- 예산 또는 채널 선택

## 6. 위임 원칙

위임 가능한 일:

- 반복 수집
- 정리
- 비교
- 초안 작성
- 시각 자료 가공
- 제목과 카피 후보 생성

사람이 해야 하는 일:

- 기준 설계
- 최종 판단
- 최종 발행 승인
- 리스크 판단
- 브랜드 방향 결정
- 법적, 윤리적 표현 검토

## 7. 운영 체크리스트

작업 시작 전:

- 목적이 명확한가?
- 어떤 팀장 역할이 주도해야 하는가?
- 필요한 원재료가 준비되어 있는가?
- 검증 기준이 있는가?

작업 중:

- 출처가 남아 있는가?
- 수치와 주장이 분리되어 있는가?
- 재사용 가능한 형태로 정리되고 있는가?
- 다음 역할자가 바로 사용할 수 있는가?

작업 완료 후:

- 오류 가능성이 표시되어 있는가?
- 대표님이 판단해야 할 항목이 분리되어 있는가?
- 발행 전 검토가 필요한 표현이 표시되어 있는가?
- 다음 액션이 명확한가?

## 8. 최종 요약

데이터팀장은 사실을 책임진다.  
컨텐츠팀장은 표현을 책임진다.  
마케팅팀장은 확산을 책임진다.  
대표님은 기준과 최종 판단을 책임진다.

이 구조를 기준으로 삼으면 작업을 맡길 때 "누가 무엇을 해야 하는지"와 "어디까지 위임해도 되는지"를 빠르게 판단할 수 있다.
