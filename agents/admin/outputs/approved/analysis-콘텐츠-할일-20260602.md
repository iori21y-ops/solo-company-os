## 2. 핵심 결론 — 콘텐츠는 "데이터 게이트"에 막혀 있다
유일한 green이 아반떼 visual 하나뿐이라, 콘텐츠가 만들 수 있는 건 (1)이미지 전용 자산과 (2)데이터 무관 범용 텍스트로 한정. 수치/가격이 들어가는 본업(스펙카드·비교표·인포그래픽·블로그 초안)은 전부 대기. **최대 병목 = 아반떼 spec-data 정합성 검증 부재.**
